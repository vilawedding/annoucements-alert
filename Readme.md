# Multi-Exchange Monitor

## Cấu trúc đã sắp xếp lại
- docs/: toàn bộ tài liệu
- scripts/tests/: script test
- scripts/examples/: ví dụ chạy thử
- scripts/legacy/: mã cũ (monolithic/legacy)
- data/: file lưu trạng thái announcement
- modules/: mã chính theo kiến trúc module

## 1) Cài đặt
```bash
npm install
```

## 2) Cấu hình
```bash
cp .env.example .env
```

## 3) Chạy bot
```bash
npm start
```

## 4) Chạy test nhanh
```bash
npm test
```
